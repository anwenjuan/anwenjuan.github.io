搭建vue脚手架（vue-cli)

发表于 2018-10-08 | 阅读次数

vue.js 是一套用于构建用户界面的渐进式框架，vue.js 两大核心思想是组件化  和数据驱动，组件可以重复利用，数据驱动释放对 DOM 的操作。vue-cli 是一个基于 vue.js 进行快速开发的完整系统。

1.安装 node.js

从node.js官网下载并安装，过程很简单，直接下一步就可以。 安装完成之后，打开命令行工具，输入node -v，出现相应的版本号，则说明安装成功。在官网下载安装 node.js 后，就自带 npm（包管理工具）了，需注意 npm 的版本最好是 3.X.X 以上，以免对后续产生影响。

2.安装 vue-cli

打开命令行工具输入：npm install vue-cli -g,安装完成之后输入vue -V（大写的 V）,出现相应的版本号，则说明安装成功。

3.用 vue-cli 构建项目

在硬盘上找到一个文件夹存放工程，在命令行工具中切换到该文件夹下，使用命令vue init webpack vuetest创建项目，vuetest是项目名称，可以随便起，但要注意不能用中文。命令输入后会进入安装阶段，需要用户输入一些信息，如下图所示：
输入完上述信息之后就开始构建项目了，项目构建完之后，可以看到工程文件夹下多出了一个项目文件夹vuetest，在命令行工具中切换  到该文件夹下，使用命令npm install安装依赖。如果安装速度太慢，可以安装淘宝镜像，npm install -g cnpm --registry=https://registry.npm.taobao.org，然后使用 cnpm 来安装。
